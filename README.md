# LaTeX to Image

A compact web-based tool for turning LaTeX math expressions into shareable images.

## What It Is
`latex2img` is a single-page web app where you can type equations in LaTeX, see an immediate preview, and export the rendered result as an image.

## Problem It Solves
Writing math for slides, notes, posts, or docs is easy with LaTeX, but quickly converting that math into clean image output is often slow and repetitive. This tool streamlines that workflow in one place.

## Main Features
- Live LaTeX preview while typing
- Export as image (copy PNG / download PNG)
- Macro panel for fast symbol/template insertion
- History panel for reusing previous equations
- Selective/local color edits on dragged/selected LaTeX text
- Global color changes when no text is selected
- Compact, interactive UI designed for fast editing

## Quick Usage
1. Open [`latex2img`](https://kwonyos.github.io/latex2img) in your browser.
2. Enter or paste your LaTeX equation in the input area.
3. Use macro buttons to insert operators, delimiters, matrices, arrows, and more.
4. Adjust color: select part of the equation to apply local color, or leave selection empty to apply a global color.
5. Export via **Copy PNG** or **Download PNG**.
